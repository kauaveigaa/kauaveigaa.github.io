1.O que é python?

  1. Falando um pouco sobre
    1. O Python é uma linguagem de programação
    2. Foi desenvolvido para ser simples, fácil de aprender e muito versátil, ou seja, você vai poder utilizar essa linguagem para diversas tarefas.
    3. Pode construir aplicativos, criar sites, desenvolver programas, criar jogos, fazer análise de dados, inteligência artificial, entre outras atividades.
  2. Características da linguagem Python
    1. Multiplataforma
    2. Multiparadigma
    3. Linguagem interpretada
  3. Para que é usado?
    1. Scripting e automação
    2. Desenvolvimento web
    3. Enquadramento de testes
    4. Big Data
    5. Inteligência artificial
    6. Computação gráfica
    7. Ciência de dados

2.Qual a origem da linguagem Python?

  1. Quem desenvolveu?
    1. Guido Van Rossum, matemático holandês
    2. Criado com o objetivo de otimizar a leitura de códigos e estimular a produtividade de quem os cria, seja este um programador ou qualquer
      outro profissional
  2. Nasceu sobre a necessidade de:
    • Economizar tempo no desenvolvimento
    • Melhorar a eficiência em um projeto desenvolvido no instituto onde Guido era pesquisador
  3. Outros
    1. Hoje, além dos desenvolvedores de software, temos biólogos, contadores, físicos e outros profissionais potencializando suas habilidades
      através dela
    2. Estes são alguns motivos que têm feito o uso do Python crescer consideravelmente nos últimos anos em detrimento de outras linguagens
